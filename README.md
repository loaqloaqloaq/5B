# 操作

プレイヤー１（開始位置：左）：

キーボード
- 移動：A/D
- ジャンプ：W
- ダッシュ：左Shift
- 武器捨てる/拾う：左Ctrl
- 攻撃：SPACE

ゲームパッド
- 移動：L　スティック
- ジャンプ：A
- ダッシュ：RB
- 武器捨てる/拾う：LB
- 攻撃：B

- ジャンプ：↑
- ダッシュ：右Shift
